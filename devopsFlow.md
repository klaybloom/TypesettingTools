### 第二阶段：编辑器稳定性修复

**问题**: `contenteditable` div 存在严重的光标定位和输入问题

**症状**:

- 光标跳动不稳定
- 换行处理异常
- 中文输入法 (IME) 兼容性差
- 文本插入位置不准确

**解决方案**:

```
contenteditable div → 原生 textarea 元素
```

**效果**: 彻底解决光标和输入稳定性问题

### 第四阶段：表格和分割线修复

**问题**: `---` 被转换为中文破折号 `——`

**根因**: `punctuation.js` 中的 `normalizeDash()` 函数

**修复**:

```javascript
// 跳过独立一行的 --- 和表格分隔行
if (/^[-*_]{3,}$/.test(trimmed)) return line;
if (/^\|.*\|$/.test(trimmed) && trimmed.includes("---")) return line;
```
